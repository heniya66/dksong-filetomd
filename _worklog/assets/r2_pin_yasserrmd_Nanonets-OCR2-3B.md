# Pin description

## 3 Pin description

**Figure 1. Pin connections**

*   **Direction of detectable acceleration (top view)**
    *   Z: Upward arrow pointing to the top of the device.
    *   Y: Red arrow pointing from left to right, representing the y-axis.
    *   X: Blue arrow pointing from bottom-left to top-right, representing the x-axis.

*   **Direction of detectable angular rate (top view)**
    *   Ω_Y: Green arrow pointing upward, representing the y-axis angular rate.
    *   Ω_R: Red arrow pointing rightward, representing the rotational rate around the z-axis.
    *   Ω_P: Blue arrow pointing downward-left to upward-right, representing the pitch angle.
