# LSM6DSM 기술 문서: 핀 설명

## 3. 핀 설명

### **그림 1. 핀 연결도**

#### **(1) 감지 가능한 가속도 방향 (상면 보기)**
- **3D 블록 구조**: 검정색 직사각형 칩
- **방향 표시**:
  - **X 축**: 파란색 화살표 (오른쪽으로 향함)
  - **Y 축**: 빨간색 화살표 (위로 향함)
  - **Z 축**: 초록색 화살표 (앞으로 향함)
- **설명**: "Direction of detectable acceleration (top view)"  
  → *감지 가능한 가속도 방향 (상면 보기)*

#### **(2) 감지 가능한 각 속도 방향 (상면 보기)**
- **3D 블록 구조**: 검정색 직사각형 칩
- **방향 표시**:
  - **Ω<sub>v</sub>**: 초록색 화살표 (위로 향함)
  - **Ω<sub>r</sub>**: 빨간색 화살표 (오른쪽으로 향함)
  - **Ω<sub>p</sub>**: 파란색 화살표 (왼쪽으로 향함)
- **설명**: "Direction of detectable angular rate (top view)"  
  → *감지 가능한 각 속도 방향 (상면 보기)*
