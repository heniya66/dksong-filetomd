## 3. 신호명/네트명 및 핀 연결 관계

### 3.1 공통 신호 구조
4개의 플래시 칩(U17~U20)은 **C1, C2, C3, C4**로 구분된 4개의 그룹으로 구성되며, 각 그룹별로 동일한 신호가 연결됩니다.

| **신호명**          | **기능**                                                                 | **연결 대상 (U17~U20)** |
|---------------------|--------------------------------------------------------------------------|-------------------------|
| `SPI_SCK_Cn`        | 시계 신호 (Clock)                                                        | SCK 핀 → [3] SPI_SCK_Cn |
| `SPI_RSTn_Cn`       | 리셋 신호 (Active Low)                                                   | RESET 핀 → [3.8] SPI_RSTn_Cn |
| `SPI_CS_Cn`         | 칩 선택 신호 (Chip Select)                                               | CS 핀 → [3] SPI_CS_Cn   |
| `SPI_D0_Cn`, ... , `SPI_D3_Cn` | 데이터 라인 (DQ0~DQ3, 4-bit)                                         | DQ0~DQ3 핀 → [3] SPI_D[3:0]_Cn |

> **참고**:  
> - `n = 1, 2, 3, 4` (U17=C1, U18=C2, U19=C3, U20=C4)  
> - `DNU_*` 핀: "Do Not Use" (미사용), 연결되지 않음.  

### 3.2 각 플래시 칩의 핀 연결 상세
#### **U17 (C1 그룹)**:
- **VCC**: B4 핀 → 3V3 (전원 공급)  
- **VIO**: E4 핀 → 3V3 (I/O 전압, 3.3V)  
- **SCK**: A2 핀 → `SPI_SCK_C1`  
- **RESET**: B2 핀 → `SPI_RSTn_C1`  
- **CS**: C2 핀 → `SPI_CS_C1`  
- **DQ0 (SO/IO1)**: D3 핀 → `SPI_D0_C1`  
- **DQ1 (SI/IO0)**: D4 핀 → `SPI_D1_C1`  
- **DQ2 (WP/IO2)**: C4 핀 → `SPI_D2_C1`  
- **DQ3 (HOLD/IO3)**: B3 핀 → `SPI_D3_C1`  

#### **U18 (C2 그룹), U19 (C3 그룹), U20 (C4 그룹)**:
- **동일한 연결 구조**를 가지며, 신호명만 `Cn`이 변경됩니다.  
  (예: U18 → `SPI_SCK_C2`, `SPI_RSTn_C2`, ...)

## 4. 설계 가이드라인 (Design Guidelines)

### 4.1 길이 매칭 (Length Matching)
- **4개의 플래시 장치 간 신호 라인 길이 동일화**가 필수적입니다.  
  - `SPI_CLK_C1~C4` (시계 신호)  
  - `SPI_CSn_C1~C4` (칩 선택 신호)  
  - `SPI_D[3:0]_C1~C4` (데이터 라인, DQ0~DQ3)  

> **목적**: QSPI 인터페이스의 타이밍 정확도를 보장하기 위해 모든 신호 라인이 동일한 전송 지연을 가져야 합니다.  
> - **구현 방법**: PCB 라우팅 시, 4개의 칩으로 이어지는 신호 라인 길이를 정밀하게 조정합니다.

### 4.2 기타 주요 사항
- **전원 필터링**: 각 플래시 칩의 VCC 핀에 0.1uF 캐패시터 (C29~C33)가 연결되어 노이즈를 감소시킵니다.  
- **GND 연결**: 모든 칩은 GND로 직접 접속되며, 3V3 전원은 공통으로 공급됩니다.

---

## 5. 도면 요약
- **4개의 QSPI 플래시 메모리** (S25FL512SAGBHC10)가 3V3 전원과 신호 라인을 통해 연결되어 있습니다.  
- **길이 매칭 요구사항**은 QSPI 인터페이스의 정상 동작을 위해 필수적입니다.  
- **부품 수량**: 플래시 칩 4개, 캐패시터 4개 (0.1uF/63V).  

> 이 회로는 PicoMatrix NxULTRA EVM 보드의 QSPI NOR Flash 모듈을 구현한 것으로, 고속 데이터 전송을 위한 정밀한 라우팅이 요구됩니다.
